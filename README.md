# biodata-2025-2-project1
# (서울 대중교통 데이터 분석 및 시각화)

## 1. 데이터분석 프로젝트 설명
* 이 프로젝트는 서울시 대중교통(지하철, 버스)의 혼잡도 및 이용 현황 데이터를 분석합니다.
* Streamlit을 사용하여 데이터를 시각화하고 사용자가 직접 상호작용할 수 있는 웹 대시보드를 구축했습니다.
* 서울교통공사의 지하철 혼잡도, 수송순위, 일별 시간대별 수송순위, 서울시 버스 정류소 현황, 서울시 버스정류장 위치정보 파일을 사용 했습니다.

## 2. 주제 선택 이유
* 왕복 등하교가 5시간씩 걸리다보니 집을 제외하고서 가장 많은 시간을 보낸 곳이 대중교통 이었습니다.
* 항상 많은 이용객들 때문에 불편 했는데 항상 사람이 많은가에 대해서 궁금해져서 해당 주제로 분석 하고 싶었습니다.

## 3. 데이터 분석 내용
* **사용한 데이터:**
    * 서울교통공사_지하철혼잡도정보_20250630.csv
    * 서울교통공사_역별 일별 시간대별 승하차인원 정보_20231231.csv
    * 서울교통공사_수송순위_20241231.csv
    * 서울시 정류소현황(2019~2023년).xlsx
    * 서울시버스정류소위치정보(20251103).xlsx

* **분석 내용:**
    * 지하철 호선/역별 시간대별 평균 혼잡도 시각화
    * 요일별(평일/주말) 승하차 인원 비교
    * 수송 순위 Top 10 역과 연계 버스 노선 수 비교 분석
    * 지하철 역 주변 이름이 겹치는 버스 정류장의 환승 가능 버스 노선 수 랭킹 표시
    * 원하는 버스 정류장 검색 후 노선도 조회 가능
